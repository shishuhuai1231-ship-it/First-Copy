## How it works
  
  Explain how your project works
  AND gate inputs are connected to the IN0 and IN1 and the output is connected to the OUT0
  CLK source is 10K clock frequency and is manipulated by a switch that is connected to a bumper that advances the clock
  Lastly there is an array if input switch for each input terminal to control on and off of each constant input terminak
